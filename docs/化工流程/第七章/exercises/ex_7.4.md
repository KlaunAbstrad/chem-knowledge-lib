# 习题 7.4

## 题目

苯(B)-甲苯(T)-二甲苯(X)-异丙苯(C)混合物进入精馏塔分离，进料组成：
$z_B=0.2$, $z_T=0.3$, $z_X=0.1$, $z_C=0.4$（摩尔分数）。
要求：馏出液中苯的回收率 95%；釜液异丙苯回收率大于 97%。
计算最少理论板数和全回流下的物料分配。全塔的操作压力为 100 kPa。

## 已知条件

| 组分 | 分类 | $z_i$ |
|------|:----:|:-----:|
| 苯 (B) | LK | 0.2 |
| 甲苯 (T) | — | 0.3 |
| 二甲苯 (X) | — | 0.1 |
| 异丙苯 (C) | HK | 0.4 |

分离要求：
- 苯在馏出液中回收率 95% → $d_B = 0.95 \times 0.2F = 0.19F$
- 异丙苯在釜液中回收率 > 97% → $b_C > 0.97 \times 0.4F = 0.388F$

### 相对挥发度（100 kPa）

| 组分 | $\alpha_i$ |
|------|:----------:|
| 苯 (B) | 2.50 |
| 甲苯 (T) | 1.00（基准）|
| 二甲苯 (X) | 0.42 |
| 异丙苯 (C) | 0.22 |


::: details 点击查看解答

## 求解思路

清晰分割 → 物料衡算 → 由 Antoine 方程计算各组分在操作条件下的 $K_i$ → $\alpha_{ij}$（塔顶塔底几何平均）→ Fenske 方程 → $N_{\min}$ → 校验非关键组分分配。

## 计算过程

## 计算过程

### 1. Fenske 方程求最少理论板数

设 $F = 100$ kmol/h 为基准。

**物料衡算**：

苯（LK）：$d_B = 0.95 \times 20 = 19$，$b_B = 1$
异丙苯（HK）：$b_C > 0.97 \times 40 = 38.8$，取 $b_C = 39$，$d_C = 1$

**清晰分割假设**：
- 甲苯（T）分割未知——它是中间组分
- 二甲苯（X）全部或主要在釜液

试差后，合理的分配：

$$D = 22\ \text{kmol/h},\ B = 78\ \text{kmol/h}$$

**馏出液组成**：
$$x_{D,B} = \frac{19}{22} = 0.864,\ x_{D,T} = \frac{2}{22} = 0.091,\ x_{D,X} \approx 0,\ x_{D,C} = \frac{1}{22} = 0.045$$

**釜液组成**：
$$x_{B,B} = \frac{1}{78} = 0.013,\ x_{B,T} = \frac{28}{78} = 0.359,\ x_{B,X} = \frac{10}{78} = 0.128,\ x_{B,C} = \frac{39}{78} = 0.500$$

**Fenske 方程**（LK = 苯, HK = 异丙苯）：

$$
N_{\min} = \frac{\log\left[\left(\frac{x_B}{x_C}\right)_D \left(\frac{x_C}{x_B}\right)_B\right]}{\log \alpha_{B,C}}
= \frac{\log\left[\left(\frac{0.864}{0.045}\right) \times \left(\frac{0.500}{0.013}\right)\right]}{\log(2.50/0.22)}
$$

$$
= \frac{\log(19.2 \times 38.46)}{\log 11.36} = \frac{\log 738.4}{1.055} = \frac{2.868}{1.055} = 2.72
$$

圆整：$N_{\min} = 3$（含再沸器）

### 2. 全回流下物料分配

Fenske 方程用于计算非关键组分的分配：

$$\frac{d_i}{b_i} = \alpha_i^{N_{\min}} \left(\frac{d}{b}\right)_{HK}$$

其中 $(d/b)_{HK} = 1/39 = 0.0256$

**甲苯 (T)**：
$$\frac{d_T}{b_T} = 1.0^{2.72} \times 0.0256 = 0.0256$$
$$d_T = \frac{f_T}{1 + 1/0.0256} = \frac{30}{1 + 39.06} = 0.75$$
$$b_T = 30 - 0.75 = 29.25$$

**二甲苯 (X)**：
$$\frac{d_X}{b_X} = 0.42^{2.72} \times 0.0256 = 0.064 \times 0.0256 = 0.00164$$
$$d_X = \frac{10}{1 + 1/0.00164} = \frac{10}{610.6} = 0.016$$
$$b_X = 10 - 0.016 = 9.984$$

## 答案

| 参数 | 值 |
|------|-----|
| $N_{\min}$ | **3**（含再沸器）|

全回流下物料分配：

| 组分 | $d_i$ (kmol/h) | $b_i$ (kmol/h) |
|------|:------------:|:------------:|
| 苯 (LK) | 19.00 | 1.00 |
| 甲苯 | 0.75 | 29.25 |
| 二甲苯 | 0.02 | 9.98 |
| 异丙苯 (HK) | 1.00 | 39.00 |
