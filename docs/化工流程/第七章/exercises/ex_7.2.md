# 习题 7.2

## 题目

进料组成为丙烷、异丁烷、正丁烷、异戊烷、正戊烷、正己烷和正庚烷，
流量分别为 2500、400、600、100、200、40 和 50 kmol/h，
塔的操作压力为 700 kPa。
分离要求是塔顶馏出液重关键组分异戊烷流量小于 15 kmol/h，
塔底釜液轻关键组分正丁烷流量小于 6 kmol/h，
计算所需的最少理论板数及非关键组分在塔顶和塔底的分布。

## 已知条件

| 组分 | 分类 | 进料 $f_i$ (kmol/h) |
|------|:----:|:-----------------:|
| 丙烷 (C3) | LNK | 2500 |
| 异丁烷 (iC4) | LNK | 400 |
| 正丁烷 (nC4) | **LK** | 600 |
| 异戊烷 (iC5) | **HK** | 100 |
| 正戊烷 (nC5) | HNK | 200 |
| 正己烷 (nC6) | HNK | 40 |
| 正庚烷 (nC7) | HNK | 50 |

LK = 正丁烷 (nC4), HK = 异戊烷 (iC5)

分离要求：
- 馏出液中 HK (iC5) < 15 kmol/h → $d_{HK} = 15$
- 釜液中 LK (nC4) < 6 kmol/h → $b_{LK} = 6$


::: details 点击查看解答

## 计算过程

### 1. 物料衡算（清晰分割假设）

LK (正丁烷)：$d_{LK} = 600 - 6 = 594$，$b_{LK} = 6$
HK (异戊烷)：$d_{HK} = 15$，$b_{HK} = 100 - 15 = 85$

LNK 全部进入馏出液：$d_{C3} = 2500$，$d_{iC4} = 400$
HNK 全部进入釜液：$b_{nC5} = 200$，$b_{nC6} = 40$，$b_{nC7} = 50$

### 2. 馏出液和釜液流量

$$D = 2500 + 400 + 594 + 15 = 3509\ \text{kmol/h}$$
$$B = 6 + 85 + 200 + 40 + 50 = 381\ \text{kmol/h}$$

### 3. 相对挥发度

在操作压力 700 kPa 下，各组分相对于 HK (iC5) 的相对挥发度需通过泡露点计算获得。
此处采用典型值（以异戊烷为基准 $\alpha = 1$）：

| 组分 | 近似 $\alpha_i$ | $x_D$ | $x_B$ |
|------|:--------------:|:-----:|:-----:|
| 丙烷 | ~4.5 | 0.712 | — |
| 异丁烷 | ~2.8 | 0.114 | — |
| 正丁烷 (LK) | ~2.1 | 0.169 | 0.0157 |
| 异戊烷 (HK) | 1.0 | 0.0043 | 0.223 |
| 正戊烷 | ~0.85 | — | 0.525 |
| 正己烷 | ~0.40 | — | 0.105 |
| 正庚烷 | ~0.20 | — | 0.131 |

### 4. Fenske 方程

$$
N_{\min} = \frac{\log\left[\left(\frac{594}{15}\right)_D \left(\frac{85}{6}\right)_B\right]}{\log 2.1}
= \frac{\log(39.6 \times 14.17)}{\log 2.1} = \frac{\log 561.3}{0.322} = \frac{2.749}{0.322} = 8.54
$$

圆整：$N_{\min} = 9$（包括再沸器）

### 5. 非关键组分分配

以 $(d/b)_{HK} = 15/85 = 0.1765$

**丙烷 (C3)**：
$$\frac{d_{C3}}{b_{C3}} = 4.5^{8.54} \times 0.1765 \gg 1 \rightarrow \text{全部在馏出液}$$

**异丁烷 (iC4)**：
$$\frac{d_{iC4}}{b_{iC4}} = 2.8^{8.54} \times 0.1765 \gg 1 \rightarrow \text{全部在馏出液}$$

**正戊烷 (nC5)**：
$$\frac{d_{nC5}}{b_{nC5}} = 0.85^{8.54} \times 0.1765 = 0.256 \times 0.1765 = 0.045 \ll 1$$
$$d_{nC5} = \frac{f_{nC5}}{1 + b_{nC5}/d_{nC5}} = \frac{200}{1 + 1/0.045} = 8.6\ \text{kmol/h}$$
$$b_{nC5} = 200 - 8.6 = 191.4\ \text{kmol/h}$$

## 答案

| 参数 | 值 |
|------|-----|
| 最少理论板数 $N_{\min}$ | **9**（含再沸器）|
| 馏出液流量 $D$ | 3509 kmol/h |
| 釜液流量 $B$ | 381 kmol/h |

主要非关键组分分布：

| 组分 | $d_i$ (kmol/h) | $b_i$ (kmol/h) |
|------|:------------:|:------------:|
| 丙烷 | 2500 | 0 |
| 异丁烷 | 400 | 0 |
| 正丁烷 (LK) | 594 | 6 |
| 异戊烷 (HK) | 15 | 85 |
| 正戊烷 | 8.6 | 191.4 |
| 正己烷 | ~0 | 40 |
| 正庚烷 | ~0 | 50 |
