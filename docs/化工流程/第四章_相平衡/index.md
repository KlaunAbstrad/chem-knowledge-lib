# 第四章 相平衡基础

> **学习目标**：学完本章后，你将能够——
> 1. 用 Antoine 方程计算任意温度下的纯组分饱和蒸气压
> 2. 计算理想和非理想体系的相平衡常数 $K_i$
> 3. 用泡点/露点方程迭代求解混合物的平衡温度
> 4. 用 Rachford-Rice 方程计算等温闪蒸的汽化率和两相组成
> 5. 用 Peng-Robinson 状态方程计算气相和液相的摩尔体积

> **前置知识**：本章为化工流程课程的基础章，不依赖其他章节。需要本科化工热力学基础知识（理想气体、拉乌尔定律）。

## 本章总结

### 核心计算链条

```
Antoine 方程 → P_i^sat → K_i = P_i^sat/P → α_ij = K_i/K_j
                                                      ↓
                                            泡点/露点迭代求温度
                                                      ↓
                                            Rachford-Rice 求汽化率 ψ
```

### 关键公式速查

| 公式 | 编号 | 用途 |
|------|:----:|------|
| $\ln P^{\text{sat}} = A - B/(T+C)$ | (4.5) | 饱和蒸气压 |
| $K_i = P_i^{\text{sat}} / P$ | (4.3) | 理想体系相平衡常数 |
| $K_i = \gamma_i P_i^{\text{sat}} / P$ | (4.4) | 非理想体系相平衡常数 |
| $\sum K_i x_i = 1$ | (4.11) | 泡点方程 |
| $\sum y_i / K_i = 1$ | (4.12) | 露点方程 |
| $f(\psi) = \sum \frac{(K_i-1)z_i}{1+\psi(K_i-1)} = 0$ | (4.25) | Rachford-Rice 闪蒸方程 |

### 后续章节

本章的 $K_i$ 和 $\alpha_{ij}$ 计算方法是第 7 章（精馏）和第 9 章（吸收）的基础。所有分离过程的简捷计算都依赖相平衡常数。
