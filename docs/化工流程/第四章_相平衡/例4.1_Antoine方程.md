# 例 4.1 — Antoine 方程计算饱和蒸气压

## 题目

在 378.47 K 下，分别计算苯、甲苯、对二甲苯的饱和蒸气压。Antoine 常数（$P^{\text{sat}}$ 单位：$10^5$ Pa，$T$ 单位：K，$\ln$ 形式）：

| 组分 | $A$ | $B$ | $C$ |
|------|-----|-----|-----|
| 苯 (Benzene) | 9.2806 | 2788.51 | −52.36 |
| 甲苯 (Toluene) | 9.3935 | 3096.52 | −53.67 |
| 对二甲苯 (p-Xylene) | 9.4761 | 3346.65 | −57.84 |

> **注意**：此处 Antoine 常数为 $\ln P^{\text{sat}} = A - \frac{B}{T + C}$ 形式，$P^{\text{sat}}$ 单位为 $10^5$ Pa。实际工程中 Antoine 常数有多种形式和单位，**使用前必须确认**。

## 求解

### 苯

$$\ln P^{\text{sat}}_1 = 9.2806 - \frac{2788.51}{378.47 - 52.36} = 9.2806 - \frac{2788.51}{326.11} = 9.2806 - 8.5507 = 0.7299$$

$$P^{\text{sat}}_1 = e^{0.7299} \times 10^5 = 2.0746 \times 10^5\ \text{Pa} = \mathbf{207461\ \text{Pa}}$$

### 甲苯

$$\ln P^{\text{sat}}_2 = 9.3935 - \frac{3096.52}{378.47 - 53.67} = 9.3935 - \frac{3096.52}{324.80} = 9.3935 - 9.5336 = -0.1401$$

$$P^{\text{sat}}_2 = e^{-0.1401} \times 10^5 = 0.8693 \times 10^5\ \text{Pa} = \mathbf{86925\ \text{Pa}}$$

### 对二甲苯

$$\ln P^{\text{sat}}_3 = 9.4761 - \frac{3346.65}{378.47 - 57.84} = 9.4761 - \frac{3346.65}{320.63} = 9.4761 - 10.4377 = -0.9616$$

$$P^{\text{sat}}_3 = e^{-0.9616} \times 10^5 = 0.3823 \times 10^5\ \text{Pa} = \mathbf{38227\ \text{Pa}}$$

## 结果

| 组分 | $P^{\text{sat}}$ (Pa) |
|------|----------------------:|
| 苯 | 207461 |
| 甲苯 | 86925 |
| 对二甲苯 | 38227 |

## 工程要点

1. **Antoine 常数有版本差异**：不同手册可能给出不同数值，同一组分可能有 2~3 套参数对应不同温度区间
2. **单位必须匹配**：本例题中 $A, B, C$ 对应于 $\ln$ 和 $10^5$ Pa，若手册给出 $\log_{10}$ 和 mmHg 的参数，需先转换
3. **温度单位**：Antoine 方程中的 $T$ 通常使用 K 或 °C，由常数 $C$ 的值可判断（$C \approx -273$ 时通常为 °C）
