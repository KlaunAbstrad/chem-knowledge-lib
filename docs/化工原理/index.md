# 化工原理（下）

> 📚 知识库核心课程

化工原理是化学工程的基础理论课程，主要研究化工生产中的三大传递过程：**动量传递**、**热量传递**和**质量传递**。本课程分为上册（流体力学、传热学）和下册（蒸馏、吸收、萃取、干燥等分离操作）。

本知识库基于下册教材编写，涵盖传质分离操作的核心内容。

---

## 课程内容

### 第六章 蒸馏

精馏是分离液体混合物最常用的方法，利用各组分挥发度差异实现分离。

| 小节 | 内容 | 状态 |

| 6.4 | 多组分精馏 | 🚧 建设中 |
| 6.5 | 特殊精馏 | 📋 规划中 |

### 第七章 吸收与解吸

气液传质操作，利用气体组分在液体中的溶解度差异进行分离。

| 小节 | 内容 | 状态 |
|:---:|------|:----:|
| 7.1 | 吸收操作 | 📋 规划中 |

### 第八章 萃取

液液萃取，利用组分在两种互不相溶溶剂中的溶解度差异进行分离。

| 小节 | 内容 | 状态 |
|:---:|------|:----:|

### 第九章 干燥

热质同时传递过程，用热能使物料中的水分蒸发脱除。

## 学习路径

```
流体力学基础 → 传热学基础 → 传质学基础 → 蒸馏 → 吸收 → 萃取 → 干燥
```

化工原理为化工流程设计提供理论基础：
- **蒸馏** ← → 化工流程·第七章 多组分精馏
- **吸收** ← → 化工流程·第九章 吸收与解吸
- **干燥** ← → 化工流程·第五章 换热器设计
