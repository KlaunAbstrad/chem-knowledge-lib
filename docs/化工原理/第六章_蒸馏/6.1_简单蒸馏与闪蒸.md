# 6.1 简单蒸馏与闪蒸

## 定义/概述

简单蒸馏（又称微分蒸馏）和闪蒸是两种基本的单级分离操作，利用部分汽化或部分冷凝实现组分分离。两者的本质区别在于：**简单蒸馏**中汽相连续引出并移走，而**闪蒸**（又称平衡蒸馏）是一次性达到气液平衡后分离。

## 简单蒸馏

### 基本原理

简单蒸馏的物料衡算（瞬时）：

$$dV = dL = -dF$$

组分的瞬时变化：

$$y = \frac{-d(Lx)}{dV} = \frac{-L dx}{V}$$

对于二元体系，写成微分形式：

$$\frac{dx}{dy} = -\frac{V}{L}$$

积分得到 **Rayleigh 方程**：

$$\ln\frac{x_F}{x_W} = \frac{1}{\alpha - 1}\left(\ln\frac{x_F}{x_W} + \alpha\ln\frac{1-x_W}{1-x_F}\right) \tag{6.1}$$

### 操作线与釜液组成

简单蒸馏的瞬时汽相组成与液相组成的关系由相平衡决定：

$$y = \frac{\alpha x}{1 + (\alpha - 1)x}$$

联立 Rayleigh 方程，可得到任意时刻釜液组成与馏出液总组成的关系。

### 工程应用

简单蒸馏适用于：
- **初步分离**：只要求初步浓缩而不需要高纯度产品
- **间歇操作**：小批量、多品种的精细化工生产
- **预分离**：作为精馏塔的预处理步骤

## 闪蒸（平衡蒸馏）

### 定义

闪蒸是指物料突然减压到低于其泡点压力，部分液体汽化并与剩余液体处于平衡状态，然后气液分离的过程。

### 操作特征

- **一次性平衡**：闪蒸过程一次性完成气液平衡，不再逐级接触
- **恒压过程**：闪蒸压力恒定，不考虑压降
- **温度变化**：物料在闪蒸过程中温度降低（吸热）

### 物料衡算

对闪蒸器做总物料衡算：

$$F = V + L,\quad Fz_F = Vy + Lx \tag{6.2}$$

组分衡算：

$$Fz_F = V y + L (z_F - V y / L) \tag{6.3}$$

### 相平衡关系

闪蒸的气液组成满足相平衡：

$$y_i = K_i x_i$$

其中 $K_i$ 为相平衡常数，与温度、压力、物系性质有关。

### 闪蒸计算步骤

1. **已知**：进料组成 $z_F$、进料状态（q）、闪蒸压力 $P$、闪蒸温度 $T$
2. **求解**：气相分率 $V/F$、气相组成 $y_i$、液相组成 $x_i$

**等温闪蒸迭代法**（已知 P，求 T）：

- 假设温度 $T$ → 计算 $K_i(T)$ → 计算相平衡常数
- 由 $y_i = K_i x_i$ 和物料衡算迭代求解
- 收敛条件：$\sum(y_i - x_i) = 0$（对理想体系）

### 非理想体系

对于非理想体系，需要引入**逸度系数**或**活度系数**修正：

$$K_i = \frac{\gamma_i \phi_i^{sat} P_i^{sat}}{\phi_i P}$$

或使用状态方程（Peng-Robinson）直接计算。

## 简单蒸馏 vs 闪蒸

| 特征 | 简单蒸馏 | 闪蒸 |
|:----:|:-------:|:----:|
| 操作方式 | 连续微分汽化 | 一次性平衡 |
| 产物 | 顶产物（累计）+ 釜残液 | 气相产物 + 液相产物 |
| 分离效果 | 较低，适合粗分离 | 中等，适合预分离 |
| 能耗 | 较高（持续供热） | 较低（减压汽化） |
| 适用场景 | 中间产物初步浓缩 | 分离要求不高的体系 |

## 关键参数

| 符号 | 含义 | 单位 |
|------|------|------|
| $F$ | 进料流量 | kmol/h |
| $V$ | 气相流量 | kmol/h |
| $L$ | 液相流量 | kmol/h |
| $z_F$ | 进料组成（摩尔分率） | 无量纲 |
| $y$ | 气相组成（摩尔分率） | 无量纲 |
| $x$ | 液相组成（摩尔分率） | 无量纲 |
| $\alpha$ | 相对挥发度 | 无量纲 |
| $K_i$ | 相平衡常数 | 无量纲 |
| $P$ | 系统压力 | kPa |
| $T$ | 温度 | K |
| $q$ | 进料热状态分数 | 无量纲 |

## 后续应用

闪蒸计算是精馏塔设计的基础：
- **进料状态**：闪蒸计算用于确定进料的热状态（q 值）
- **多级接触**：精馏塔的每一块理论板都是一个闪蒸过程
- **分离序列**：闪蒸常作为多组分分离序列的第一步
