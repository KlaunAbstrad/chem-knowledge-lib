# 习题 6.4

## 题目

某多组分精馏塔分离 C3（丙烷）、iC4（异丁烷）、nC4（正丁烷）、iC5（异戊烷）混合物。进料组成：$f_{C3} = 5$，$f_{iC4} = 25$，$f_{nC4} = 45$，$f_{iC5} = 25$ kmol/h（总计 100 kmol/h）。

关键组分选择：C3（轻关键组分 LK），nC4（重关键组分 HK）。

分离要求：馏出液中 C3 回收率 95%，釜残液中 nC4 回收率 95%。

求：清晰分割假定下的物料衡算（$D, B, x_{Di}, x_{Bi}$）。

---

::: details 点击查看解答

## 计算过程

### 1. LNK 和 HNK 确定

根据挥发度顺序：C3 > iC4 > nC4 > iC5

- LNK = iC4（比 LK 更轻）
- HNK = iC5（比 HK 更重）

### 2. 清晰分割假定

LNK 全部进馏出液，HNK 全部进釜残液：

$$d_{iC4} = f_{iC4} = 25,\quad b_{iC4} = 0$$

$$d_{iC5} = 0,\quad b_{iC5} = f_{iC5} = 25$$

### 3. 关键组分衡算

C3 回收率 95%：
$$d_{C3} = 0.95 \times 5 = 4.75,\quad b_{C3} = 0.25$$

nC4 回收率 95%：
$$b_{nC4} = 0.95 \times 45 = 42.75,\quad d_{nC4} = 2.25$$

### 4. 总物料衡算

$$D = d_{C3} + d_{iC4} + d_{nC4} = 4.75 + 25 + 2.25 = 32\ \text{kmol/h}$$

$$B = b_{C3} + b_{nC4} + b_{iC5} = 0.25 + 42.75 + 25 = 68\ \text{kmol/h}$$

校验：$D + B = 32 + 68 = 100 = F$ ✓

### 5. 产品组成

| 组分 | $d_i$ | $x_{D,i}$ | $b_i$ | $x_{B,i}$ |
|------|:-----:|:---------:|:-----:|:---------:|
| C3 (LK) | 4.75 | 0.148 | 0.25 | 0.004 |
| iC4 (LNK) | 25 | 0.781 | 0 | 0 |
| nC4 (HK) | 2.25 | 0.070 | 42.75 | 0.629 |
| iC5 (HNK) | 0 | 0 | 25 | 0.368 |
| **合计** | **32** | **1.000** | **68** | **1.000** |

## 答案

馏出液流量 $D =$ **32 kmol/h**，釜残液流量 $B =$ **68 kmol/h**。
