# 习题 6.2

## 题目

判断能否用闪蒸分离苯-甲苯混合物。已知进料 $z_F = 0.5$，操作压力 $P = 202.6$ kPa（2 atm），相对挥发度 $\alpha = 2.5$。

闪蒸条件：要求气相分率 $V/F = 0.3$，且 $x_A \leq 0.3$。

---

::: details 点击查看解答

## 分析

判断闪蒸是否可行的关键是：给定操作条件下，能否达到指定的分离要求。

## 计算过程

### 1. 闪蒸方程

由物料衡算和相平衡：
$$0.5 = 0.3 \cdot \frac{2.5x}{1+1.5x} + 0.7x$$

化简：
$$0.3 \cdot \frac{2.5x}{1+1.5x} = 0.5 - 0.7x = 0.5 - 0.7x$$

$$0.75x = (0.5 - 0.7x)(1 + 1.5x) = 0.5 + 0.75x - 0.7x - 1.05x^2$$

$$0.75x = 0.5 + 0.05x - 1.05x^2$$

$$1.05x^2 - 0.7x + 0.5 = 0$$

### 2. 求解二次方程

$$x = \frac{0.7 \pm \sqrt{0.49 - 2.1}}{2.1} = \frac{0.7 \pm \sqrt{-1.61}}{2.1}$$

**判别式 < 0，无实解** → 闪蒸无法达到 $V/F = 0.3$ 且 $x_A \leq 0.3$ 的条件。

### 3. 分析原因

要满足 $x_A \leq 0.3$，闪蒸程度需要足够大。但给定 $\alpha = 2.5$ 和 $z_F = 0.5$，要使 $x_A \leq 0.3$，需要更高的气相分率。

## 答案

**不能**。给定条件下无法同时满足 $V/F = 0.3$ 和 $x_A \leq 0.3$ 的要求。
