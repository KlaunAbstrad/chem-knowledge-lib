# 第六章 蒸馏

## 学习目标

学完本章后，你将能够：
1. 理解简单蒸馏、闪蒸、精馏的区别和应用场景
2. 掌握 McCabe-Thiele 图解法（逐板计算）的基本原理
3. 应用 Fenske 方程计算最小理论板数
4. 应用 Underwood 方程计算最小回流比
5. 理解多组分精馏的关键组分选择原则
6. 能够对二元精馏塔进行工艺计算（塔板数、回流比、进料位置）

## 前置知识

学习本章前需掌握：
- **流体力学基础**：伯努利方程、流体阻力（→ 上册内容）
- **传热学基础**：冷凝与沸腾（→ 上册内容）
- **相平衡基础**：相对挥发度 α、相平衡常数 K（→ 化工流程·第四章）

## 本章总结

### 核心计算链条

```
进料状态 q → 进料线方程 → q线与精馏线交点
            → α_LK/HK → Fenske → N_min
            → Underwood → R_min
            → Gilliland → 实际塔板数 N
```

### 关键公式速查

| 公式 | 名称 | 编号 |
|------|------|:----:|
| $y = \frac{\alpha x}{1 + (\alpha - 1)x}$ | 相平衡关系 | 6.1 |
| $q = \frac{C_{p,L}(T_b - T_F)}{r_F}$ | 进料热状态 | 6.2 |
| $R_{\min} = \sum\frac{\alpha_i x_{D,i}}{\alpha_i - \theta} - 1$ | Underwood 方程 | 6.3 |
| $N_{\min} = \frac{\log\left(\frac{x_L}{x_H}\right)_{D,HK}}{\log\alpha_{LK/HK}}$ | Fenske 方程 | 6.4 |

### 常见陷阱

1. **Antoine 方程温度单位**：T 必须用 K，不能用 °C
2. **q 线方程的写法**：不同进料状态（泡点、冷液、汽液混合、过热蒸汽）q 值不同
3. **回流比选择**：实际回流比通常为 1.2~1.5 倍最小回流比，过大不经济，过小效率低
4. **塔板效率**：实际塔板数 = 理论塔板数 / 塔板效率
